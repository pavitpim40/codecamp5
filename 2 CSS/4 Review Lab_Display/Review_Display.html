Question Answer
1. Value ใน display property มีอะไรบ้าง
  • inline : แสดงผลเป็น element ที่อยู่ในแถว (บรรทัด) สูญเสียคุณสมบัติด้านความสูงและความกว้าง
  • block : แสดงผลเป็น block จะใช้ความกว้างทั้งหมดที่สามารถใช้ได้ และขึ้นบรรทัดใหม่ให้อัตโนมัติ
  • inline block : แสดงผลแบบผสมผสาน กล่าวคือเป็น inline แต่สามารถปรับความกว้างและความสูงแบบ block ได้
  • contents : ทำให้สภาพ container หายไป และทำให้ child element ขึ้นมาเป็น element หลักในไฟล์ html
  • flex : แสดงผลแบบ block แต่สามารถใช้ flex property ปรับค่าได้
  • grid : แสดงผลแบบ block แต่สามารถใช้ grid property ในการปรับค่าได้
  • inline-flex : แสดงผลเป็น inline แต่นับเป็น flex container
  • inline-grid :แสดงผลเป็น inline แต่นับเป็น grid container
  • inline-table :แสดงผลเป็น inline แต่นับเป็น grid container
  • list-item : แสดงผลเป็น tag list
  • run-in : แสดงผลเป็น block หรือ inline อย่างใดอย่างหนึ่ง ขึ้นอยู่กับบริบท
  • table : แสดงผลเป็นตาราง
  • table-caption : แสดงผลเหมือนกับคำอธิบายตาราง
  • table-column-group :แสดงผลเหมือน tag <colgroup>
  • table-header-group :แสดงผลเหมือน tag <thead>
  • table-footer-group : แสดงผลเหมือน tag <tfoot>
  • table-row-group : แสดงผลเหมือน tag <tbody>
  • table-cell : แสดงผลเหมือน tag <td>
  • table-column :แสดงผลเหมือน tag <col>
  • table-row :แสดงผลเหมือน tag <tr>
  • none : ไม่แสดงผล
  • initial : ตั้งค่าให้ element เป็นค่าตั้งต้น
  • inherit : ใช้ค่า display property จาก parent element
2.Grid และ Flex ต่างกันอย่างไร Grid จะสามารถจัดการกับ element ในสองแนวพร้อมๆกันได้ดี ( row and column )
Flex จะสามารถจัดกการกับ element ในหนึ่งมิติได้ดีกว่า (row or colum)
3.Justify กับ Align ต่างกันอย่างไร justify : คือการจัด element ในแนวนอน
align : คือการจัด element ในแนวตั้ง
** ถ้ามีการสลับทิศทางจาก row เป็น column คำสั่ง justify กับ align จะสลับกัน
