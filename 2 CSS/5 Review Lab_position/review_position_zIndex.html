Question Answer
1.Position คืออะไรและมีอะไรบ้าง position คือการกำหนดคุณสมบัติของ element ในด้านของการจัดตำแหน่ง element ประกอบไปด้วย
  • statics : ไม่ทำการจัดตำแหน่งใดๆเป็นพิเศษ ตัว element จะไหลตาม normal flow ปกติ
  • realative : สามารถจัดตำแหน่งได้โดยอ้างอิงจากตำแหน่งปกติเดิมของ element
  • fixed : ทำให้ element ถูกยึดตำแหน่งตายตัวบนหน้าจอ
  • absolute : เป็นการจัดตำแหน่งโดยอ้างอิงจากตัว element ที่เหนือกว่า (ancestor) อันที่ใกล้ที่สุดที่มีการใช้คุณสมบัติ position
    โดยคุณสมบัติของ ancestor นั้นเป็นอะไรก็ได้ที่ไม่ใช่ statics
    และหากไม่มี ancestor ให้อ้างอิง จะทำการอ้างอิงตำแหน่งจากด้านซ้ายบนของหน้าจอแสดงผลแทน
  • sticky : จะคงคุณสมบัติ relative ไว้จนกว่าจะถึงค่า offset ที่กำหนด จากนั้นจะแสดงผลเป็น fixed
2. z-index คืออะไร ใช้อย่างไร ? z-index คือตัวลำดับ layer ของ element
ใช้โดยการกำหนดค่าเป็นจำนวนเต็ม เช่น -1,0,1 หากตัวเลขมีค่ามากสุดก็จะอยู่ layer บนสุด (user เห็นแน่นอน)
หากตัวเลขน้อยสุดก็จะทำให้ element นั้นอยู่ด้านหลังสุด ซึ่งหากมี element ใดมาบังก็อาจจะทำให้มองไม่เห็นหรือเห็นได้ไม่ครบ
