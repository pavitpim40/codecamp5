Question Answer
1.CSS syntax ประกอบด้วยอะไรบ้าง Syntax ประกอบไปด้วย 2 ส่วน คือ
1.Seclector : สำหรับประกาศว่าต้องการให้ CSS rule ที่กำลังจะเขียนขึ้น ถูกนำไปใช้ที่ element ใด และ/หรือ สถานะใด
2.Declaration : เป็นส่วนที่ใช้สำหรับเขียนรายละเอียดและกฏต่างๆที่จะใช้ในการตกแต่ง ประกอบด้วย
2.1 Property : คุณสมบัติทีต้องการตกแต่ง เช่น background-color , text-align เป็นต้น
2.2 Value : เป็นค่าที่เรากำหนดสำหรับการตกแต่ง เช่น สำหรับ font-size จะมี value เป็น 30px หรือ 3rem เป็นต้น ขึ้นอยู่กับ Property ที่เราใช้
2.การเขียน CSS มีกี่แบบ อะไรบ้าง มีทั้งหมด 3 แบบ
1.Inline CSS : ตกแต่งโดยการเขียน CSS rule ลงใน element ของ HTML โดยตรง (มี Specificity สูงสุด)
2.Internal CSS : ตกแต่งโดยการเขียน CSS rule ไว้ที่ส่วน head ของเอกสาร HTML ภายใต้ tag style
3.External CSS : ตกแต่งโดยการเขียน CSS rule ไว้ที่อีกไฟล์หนึ่งซึ่งเป็นไฟล์สกุล .CSS และใช้การลิงค์มายังเอกสาร HTML ที่ต้องการ
3.Seclector มีกี่แบบ อะไรบ้าง มีทั้งหมด 7 ประเภท
1.Universal Seclector (*) ใช้ตกแต่งทุก element ใน HTML
2.Element Selector (eg. p, nav) ใช้ตกแต่งโดยการเลือกจากประเภทของ element
3.Class Selector (.) ใช้ตกแต่งโดยการเลือกจากชื่อของ class
4.Id Seclector (#) ใช้ตกแต่งโดยการเลือกจากชื่อของ id
5.Psuedo-class Selector (eg. div:hover) ใช้ตกแต่งโดยการเลือกจากสถานะของ element
6.Psuedo-Element Selector (eg. p::after) ใช้ตกแต่งเฉพาะบางส่วนของ element
7.Attribute Selector : ใช้ตกแต่งโดยเลือกจาก Attribute หรือ value

หากแบ่งตามลักษณะการประกาศ มีทั้งหมด 3ประเภท
1.Simple Selector : ใช้ Seclector อย่างใดอย่างหนึ่งจาก 7 ประเภทก่อนหน้านี้
2.Combinator Selector (eg. div p) ใช้ตกแต่งโดยการเลือกจากสิ่งที่ประกาศ โดยพิจารณาความสัมพันธ์ของสิ่งที่ประกาศร่วมด้วย ทำให้ตกแต่งได้specific มากขึ้น
3.Grouping Selector (,) ใช้ตกแต่งหลายๆ selector พร้อมๆกัน
4.Combinator มีกี่แบบ อะไรบ้าง มีทั้งหมด 4 แบบ
1.Descendant : การตกแต่งจะส่งผลต่อ selector ตัวหลังที่ระดับต่ำกว่า ลงไปทุกลำดับขั้น
2.Child : การตกแต่งจะส่งผลต่อ selector ตัวหลังที่ระดับต่ำกว่า ลงไปเพียงลำดับขั้นเดียว
3.Adjacent Sibling : การตกแต่งจะส่งผลต่อ selector ตัวหลัง ที่อยู่ระดับเดียวกันถัดไปเพียงตัวเดียว
4.General Sibling : การตกแต่งจะส่งผลต่อ selector ตัวหลัง ที่อยู่ระดับเดียวกันถัดไปทุกตัว
