Question Answer
1.Property และ Value คืออะไร แตกต่างกันอย่างไร - Property คือตัวกำหนดคุณสมบัติต่างๆที่เราต้องการปรับแต่งในเอกสาร HTML เช่น border-color,display เป็นต้น
- Value คือค่าของแต่ละ property ไม่จำเป็นต้องเป็นตัวเลขอย่างเดียว
   เช่น display : flex ; (display คือ property , flex คือ value)
- จุดแตกต่างคือ ตัวหนึ่งคือคุณสมบัติ ส่วนอีกตัวคือ"ค่า"ของคุณสมบัตินั้นๆ
2.Property ที่สอนใน Precamp มีกี่กลุ่ม อะไรบ้าง มีทั้งหมด 12 กลุ่ม (รวมทั้งจาก HTML และ CSS)
1.Background Properties
2.Border Properties
3.Color Properties
4.Dimension Properties เช่น width,height
5.Font Properties
6.Margin Properties
7.Padding Properties
8.Table Properties เช่น border-collapse,caption
9.Text Properties
10.Flexible Box Layout
11.Grid Layout
12.Visual formatting Properties : property ที่เกี่ยวข้องกับการจัดหน้า เช่น display , z-index , positon
3.Value unit มีกี่ประเภท อะไรบ้าง สามารถแบ่งออกได้ 2 ประเภท
1.Absolute Length Unit : ความยาวสัมบูรณ์ (ความยาวตามสเกลตามจริง) ได้แก่
  • cm : centimeters
  • mm millimeters
  • in inches (1in = 96px = 2.54cm)
  • px * pixels (1px = 1/96th of 1in)
  • pt points (1pt = 1/72 of 1in)
  • pc picas (1pc = 12 pt)
2.Relative Length Unit : ความยาวสัมพัทธ์ (กำหนดความยาวโดยมีค่าความยาวอื่นเป็นตัวอ้างอิง) ได้แก่
  • em : คิดความยาวเป็นจำนวนเท่าของ font-size
  • ex : คิดความยาวเป็นจำนวนเท่าของ font-size โดยวัดระยะตามแนวนอน
  • ch : คิดความยาวเป็นจำนวนเท่าของความกว้างเลข "0"
  • rem : คิดความยาวเป็นจำนวนเท่าของ root font-size (หากไม่มีการกำหนดจะเป็น 16px by defualut)
  • vw : คิดความยาวเป็น 1% ของความกว้าง viewport (ขึ้นอยู่กับอุปกรณ์ที่ user ใช้)
  • vh : คิดความยาวเป็น 1% ของความสูง viewport (ขึ้นอยู่กับอุปกรณ์ที่ user ใช้)
  • vmin : คิดความยาวเป็น 1% จากด้านที่เล็กกว่า
  • vmax : คิดความยาวเป็น 1% จากด้านที่ยาวกว่า
  • % : คิดระยะจาก parent element (เช่น width : 50% คือ 100px หาก parent element กว้าง 200px)
4. Vulue ที่นิยมใช้ (คำถามเสริม) ฺValue ที่นิยมใช้กันได้แก่
  • px
  • em
  • rem (for Responsive website)
  • vw,vh (for Responsive website)
  • %
