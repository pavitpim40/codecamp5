3. แท็ก<b>, <i> ต่างจากแท็ก <strong> , <em> หรือไม่ อย่างไร

3.1 Ans : ระหว่างแท็ก<b> กับ แท็ก<strong>

เหมือนกัน : ตอนแสดงผลจะทำให้ตัวอักษรหนาขึ้น
ต่างกัน : แท็ก<strong > จะมีผลต่อความสำคัญของคำนั้น เช่นเวลามีการอ่านออกเสียงผ่านโปรแกรมต่างๆ คำที่มีการใช้ แท็กstrong จะออกเสียงต่างจากตัวอื่น
หรือในบาง gadget เช่น Palm Pilot ทื่ไม่สามารถแสดงผลให้ตัวหนาขึ้นได้ก็จะแสดงผลเป็นการขีดเส้นใต้แทน เพื่อแสดงถึงระดับความสำคัญที่มากกว่าคำอื่นๆ
ในขณะที่ แท็ก <b> เป็นเพียงการตกแต่ง style ให้เส้นหนาขึ้นเท่านั้น

3.2 Ans : ระหว่างแท็ก<i> กับ แท็ก<em>

เหมือนกัน : ตอนแสดงผลจะทำให้ตัวอักษรเอียง
ต่างกัน : แท็ก<strong > จะมีผลต่อการเน้นเสียงของคำนั้น และอาจทำให้ความหมายเปลี่ยนในบางบริบท เช่น แมวเป็นสัตว์ที่น่ารัก
ประโยคข้างต้นอาจใช้ในการโต้เถียงกับอีกคนที่บอกว่าแมวเป็นสัตว์ที่น่าเกลียด(ด้วยการเน้นเสียง)
ในขณะที่ แท็ก <i > เป็นเพียงการตกแต่ง style เท่านั้น ถ้าไม่ต้องการเน้นเสียงก็ให้ใช้แท็กนี้

4.หา tag ที่สามารถแสดงข้อความกลับหลังได้

Ans : ใช้ tag < bdo > โดยมี attribute เป็น dir และvalue เป็น rtl รอบข้อความที่ต้องการให้กลับหลัง
ตัวอย่างเช่นถ้าพิมพ์ว่า < bdo dir="rtl"> hello </bdo>จะแสดงผลเป็น hello
